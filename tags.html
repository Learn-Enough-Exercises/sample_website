Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML >tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
td table data include a table data cell
div block section creates a block section to separate different content
header header section creates a header section for navbar etc.
ol ordered list creates a section for an ordered list
ul unordered list creates a section for an unordered list
li list creates a list element
br line break breaks a line in document

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> engine goes brrr...
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code syntax show source code <code>This is a source code</code> Hi this is a code tag
span span makes an inline section <span>Anything inside</span> Anything inside
blockquote blockquote quotes text <blockquote>Fear no one!</blockquote>
Fear no one!

Document Tags

Tag Name Purpose
html HTML Specifier Initiates the html page
head Header Section Info about the document including title are mentioned here
title Title Tag title of page is mentioned here
meta Meta Tag Information about document like character set are mentioned here
body Body Section Initiates the body of the document where all content visible to us is mentioned
comment Comment tag Adds a comment which is not included in code
